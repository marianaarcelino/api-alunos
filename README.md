# 📚 API de Alunos

Projeto simples desenvolvido com Node.js e Express para listar alunos.

## 🚀 Como executar

```bash
npm install
node App.js
```

## 📡 Endpoint

GET http://localhost:3000/api/alunos/

Retorna a lista de alunos em JSON.

## 📁 Estrutura

* models
* repositories
* services
* routers
